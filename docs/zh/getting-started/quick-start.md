# 使用说明

---

## 启动

```bash
uv run streamlit run web/app.py
# 或启动 API 服务
uv run uvicorn api.app:app --host 0.0.0.0 --port 8000
```

---

## 页面导航

| 页面 | 功能 |
|------|------|
| 🎬 Home | 视频生成主页，选择管线、配置参数、一键生成 |
| 📚 History | 历史记录，查看/下载/复制/删除已生成的视频 |

## 视频生成管线

### 快速创作
输入一个主题 → AI 自动完成文案/配图/配音/合成。

### 自定义素材
上传自己的照片/视频 → AI 分析素材 → 自动配文案和配音。

### 数字人口播
上传形象图片 + 商品图片 → 输入文案 → 生成数字人带货视频。

### 图生视频
上传首帧图片 + 提示词 → AI 生成动态视频。

### 动作迁移
上传参考视频 + 图片 → AI 迁移动作到图片人物。

---

## 高级功能

### 字幕
侧边栏「高级视频设置」→ 开启「自动字幕」→ 支持顶部/底部位置。

### 转场特效
选择帧间过渡动画：无/交叉淡入淡出/黑场/左滑/放大。

### 多角色配音
为不同帧交替使用不同音色，营造对话感。

### 批量生成
输入多个主题（每行一个）→ 一键批量生成所有视频。

### 链接转视频
粘贴文章/博客 URL → AI 提取内容 → 自动生成视频摘要。

### 视频编辑
在 Editor 页面：选择已完成任务 → 调整帧顺序 → 裁剪时长 → 重新生成单帧 → 导出 ZIP。

### 脚本模板
Home 页面可选 10 种预设模板：新闻/教程/测评/故事/冥想/科技/美食/旅行/健身/财经。

### AI 音乐
音乐风格智能推荐，10 种风格可选（电影感/LoFi/电子/原声/爵士/摇滚/氛围/嘻哈/古典/商务）。

### A/B 测试
同一主题多版本对比：不同风格/转场/配音/质量/字幕组合。

## API 使用

```bash
# 同步生成
curl -X POST http://localhost:8000/api/video/generate/sync \
  -H "Content-Type: application/json" \
  -d '{"text": "如何提高工作效率", "mode": "generate", "n_scenes": 5}'

# 异步 + WebSocket 实时进度
curl -X POST http://localhost:8000/api/video/generate/async \
  -d '{"text": "AI的未来", "mode": "generate"}'
# 返回 task_id，连接 ws://localhost:8000/api/ws/progress/{task_id}
```
